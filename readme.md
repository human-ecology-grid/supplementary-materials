These are supplementary materials for the paper 

Taras Zakharko, Anna Graff, Balthasar Bickel. The Human Ecology Grid: A Comprehensive Multi-Domain Global Spatial Dataset for Environmental, Ethnolinguistic, and Socioeconomic Research.

## Instructions

The supplementary materials contain R scripts and plots used in the paper. You need the [`humanEcologyGrid` package](https://github.com/human-ecology-grid/R-package-humanEcologyGrid) to run the scripts. 

- `generate_maps.R` generates the data maps used in the paper and places them in `plots/`
- `run_case_study1.R` runs the first case study (correlation between per-cell GDP and nighttime lights)
- `run_case_study2.R` runs the second case study (correlation between per-cell ethnic group marginalization and language endangerment)
